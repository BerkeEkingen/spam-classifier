# Spam Classifier AI 🚀

Live Demo: https://spam-classifier-pc3r.onrender.com


A simple Machine Learning web app that detects whether a message is spam or not.

## 🚀 Demo

Run locally:

```bash
pip install scikit-learn flask
python app.py
```

Then open: http://127.0.0.1:5000

## 🧠 How it works

* Text is converted to numbers using CountVectorizer
* A Naive Bayes model classifies messages as Spam or Not Spam

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Flask

## 📌 Features

* Detects spam messages
* Simple web interface
* Fast and lightweight
